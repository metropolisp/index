Disclaimer,

This project is highly experimental. Even though we will try to take precautions to run things in a containerized environment abstracted by the rest of the system, you are granting access to something that can do arbitrary code execution on your machine. Be mindful of that and by using it, you are agreeing any and all copyright owners & contributors are not legally liable for any data loss, attackers getting access to your machine, etc. Be safe and stay smart, and make backups.

Philosophy

Communal Computing

Born from CLOG, but stripped of polite ambition. Ozu is not here to render dashboards. It is here to be a machine.

Spaced Out

Humans Operate In More That 1.5 Dimensions; So Should The Ways We Interact With Interfaces. The system state lives on the server — your browser is merely projection.

Live & Reflective

Every session is mutable. Every object inspectable. Every function redefinable in motion.

Homoiconoclasm

Code‑Should‑Be‑Data & Data‑Should‑Be‑Code Whenever Possible.

Applications Not Appliances

Everything function should be able to be called / applied to whatever you want -- not just what one explicitly makes available.

FAQs

Q: What is Ozu?

A: Ozu is a Lisp machine over the wire, projecting directly into your browser.

Q: How does it work?

A: It runs Common Lisp on the server and streams the interface to your browser.

Q: Is it open source?

A: Yes, check the GitHub repository for more details.
